# 🌐 Web Development Level-1

## 📌 Project Description

This repository contains three frontend web development projects developed using **HTML5, CSS3, and JavaScript**. Each project is designed to strengthen core web development skills, including responsive layouts, user interface design, and interactive web applications. These projects demonstrate practical knowledge of frontend development and problem-solving.

---

## ✨ Projects Included

### 🚀 1. Landing Page
A modern and responsive landing page with an attractive layout and user-friendly design.

**Features**
- Responsive Design
- Navigation Bar
- Hero Section
- Call-to-Action Button
- Clean and Modern UI

---

### 👩‍💻 2. Portfolio Website
A personal portfolio website showcasing personal information, skills, projects, and contact details.

**Features**
- About Me Section
- Skills Section
- Projects Showcase
- Contact Information
- Responsive Layout
- Professional User Interface

---

### 🌡️ 3. Temperature Converter
A simple web application that converts temperature values between Celsius and Fahrenheit instantly.

**Features**
- Celsius to Fahrenheit Conversion
- Fahrenheit to Celsius Conversion
- Instant Result
- User-Friendly Interface
- Responsive Design
